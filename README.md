
# README

## Overview

This is a static personal portfolio and blog website for Luis Ronquillo, a Java developer. The site is hosted via GitHub Pages at `luisronquillo.com` and consists of plain HTML, CSS, and JavaScript with no build system or backend.

## Quick Start

Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 8080
```

## Key Features

- **Portfolio Projects** — Showcase of development work
- **Blog Section** — Technical articles and notes
- **About Me** — Personal introduction
- **Responsive Design** — Bootstrap v5.3.3 with Nord color palette

## Structure

- `index.html` — Main entry point
- `blog-section/blog_entry.html` — Blog entry template
- `css/style.css` — Custom styles (Nord colors)
- `js/` — Bootstrap bundle and utilities
- `resources/` — Downloadable files (CV, etc.)
- `img/` — Images and project assets

All assets are vendored locally except Bootstrap Icons (CDN).
